# セッション開始プロンプト

## 用途

Claude Code キャッチアップの写経セッションを始めるとき、このフォルダを選択した上で以下のプロンプトを貼る。
Coworkプロジェクトのインストラクションに設定してもOK。

---

## プロンプト

```
Claude Code のベストプラクティスを写経しながら学んでいます。

まず以下のファイルを読んで、現在の状況を把握してください：
1. PROGRESS.md — どこまで進んだか・次どこから
2. meta/001-folder-structure.md — フォルダ構成のルール

その上で、以下のどれかを提案してください：
- 前回の続きのセクションから写経を再開する
- 写経ノートから使えるパターンを extractions/ に抽出する
- 抽出済みパターンを compound-effect/assets/ に昇格する

ノートの作成時は meta/templates/study-note.md のテンプレートに従ってください。
ノート作成時のルール：
- 「写経」セクションはユーザー自身が書く。Claudeはテンプレート骨格と振り返り質問だけ用意する
進展があったら PROGRESS.md を更新してください。
```
